Welcome to Verkada_py's documentation!
======================================



Verkada_py is an unofficial Python Library for Verkada's REST API. Verkada's api documentation can be found at https://apidocs.verkada.com.
The goal of this library is to make it easier for developers to interact with Verkada's API. There are two main classes that are used ``Organization`` and ``Camera``.
